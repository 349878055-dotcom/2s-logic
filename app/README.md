# 2S-LOGIC | app 路由与协议区索引 (二级)

> **🎯 区域职责：** 本大区是 Next.js 的门面，专管「对外接客」（API 接收、页面路由分发），**绝对不生产核心逻辑**。

## 1. 子车间导航 (Folders)
如果你要修改具体业务，请立刻进入以下子目录，并阅读其中的 README：

- **[/api] - 自动化传送带 (后端接口)**
  - 范围：所有的 `GET/POST` 请求都在这里。
  - 禁令：只负责调度，遇到复杂计算必须调用 `src/lib/`。
- **[/admin] - 管理员后台**
  - 范围：内部审核工具（如 `seed-auditor`）。

## 2. 驻留文件纪律 (Local Files)
本目录下散落的 4 个文件是**全局门面**，Cursor 严禁擅自动这里的逻辑架构：

- **`page.tsx` (主页入口):** 它只是一个空壳！它只能把 `src/views/` 里的组装页面引进来，绝对不许在这里写状态（Store）和复杂的 UI 交互。
- **`layout.tsx` (全局骨架):** 负责挂载字体、全局 Provider。没要求绝不要动。
- **`globals.css` (全局样式):** Tailwind 基础配置层。
- **`favicon.ico`:** 网站图标，忽略即可。

## 3. ⚠️ 跨界隔离警告
- 如果你要加一个新页面，在 `app/` 建文件夹没问题，但页面的具体内容必须去 `src/views/` 写，然后在这里 `import`。
- **严禁**在此大区的任何文件中编写 11 维度 $C_{n+1}$ 的数学计算！
