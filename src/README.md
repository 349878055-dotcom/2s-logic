> **⚠️ CURSOR 强制指令：** 只要你在 `src/` 目录下活动，必须先根据本索引确定你的「逻辑大区」，严禁跨区胡乱搜索！

## 1. 区域划分 (The Four Districts)

本目录由以下四个互不隶属的大区组成：

1. **[/app] - 协议与入口区**
   - 负责：API 路由、页面入口。
   - 规则：只准接发数据，不准写核心逻辑。

2. **[/components] - UI 零件加工区**
   - 负责：雷达图、滑块、按钮等纯界面组件。
   - 规则：只准写样式，不准写算法。

3. **[/lib] - 逻辑总部 (大脑)**
   - 负责：11 维度演化算法 ($C_{n+1}$)、数据读写、状态管理。
   - 规则：全项目最高机密，代码必须极致原子化。

4. **[/views] - 业务组装车间**
   - 负责：把零件组装成功能页面。
   - 规则：只负责组装，不负责生产零件或逻辑。

## 2. 导航协议

- 收到指令后，先判断属于哪个大区。
- **进入大区文件夹后，必须立即阅读该文件夹下的 README 进一步定位。**
